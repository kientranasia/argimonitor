# Security

## Reporting a vulnerability

Please **do not** open a public issue for undisclosed security problems.

- Prefer [GitHub Security Advisories](https://github.com/kientranasia/argimonitor/security/advisories/new) for this repository, if enabled.
- Otherwise, contact the maintainers privately with enough detail to reproduce the issue (without sharing live secrets).

## Operational notes

- Never commit `.env` or live API keys. Use `.env.example` only for variable names and placeholders.
- Rotate any key that may have been copied into a ticket, chat, or accidental commit.
